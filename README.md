\# 🚀 韩山师范一键评教脚本



这是一个运行在浏览器中的 Tampermonkey（篡改猴）用户脚本，专为韩山师范学院教务系统设计。一键自动完成所有评教选项，自动填写评语，并自动提交，告别期末疯狂点鼠标的折磨！



\## ✨ 功能特点



\* \*\*一键全自动\*\*：点击悬浮按钮，自动勾选所有“优”，并将最后一个选项改为“良”（防止系统满分风控）。

\* \*\*自动写评语\*\*：自动在主观题文本框填入标准好评（如“授课清晰，非常满意”）。

\* \*\*弹窗拦截\*\*：智能拦截系统原生的确认和提示弹窗，防止提交成功后页面卡住。

\* \*\*高兼容性\*\*：支持动态加载的教务系统页面，内置智能轮询检测机制，不怕页面卡顿。



\## 🛠️ 安装教程



\*\*第一步：安装浏览器插件 (Tampermonkey)\*\*

如果你还没有安装“篡改猴”扩展，请先在你的浏览器扩展商店中安装它。

\* Edge 浏览器：在扩展商店搜索 `Tampermonkey` 并点击“获取”。

\* Chrome 浏览器：在 Chrome Web Store 搜索 `Tampermonkey` 并添加。



\*\*第二步：安装本脚本\*\*

\* \*\*一键安装\*\*：直接点击 \[https://gitee.com/sheemia/evaluate/raw/main/%E9%9F%A9%E5%B8%88%E4%B8%80%E9%94%AE%E8%AF%84%E6%95%99%E8%84%9A%E6%9C%AC.user.js] 进行安装。

\* \*\*本地安装\*\*：下载本仓库中的 `韩师一键评教.user.js` 文件，将其拖拽进已安装篡改猴的浏览器窗口中，点击“安装”即可。



\## 📖 使用说明



1\. 登录韩山师范学院教务系统 (`eams` 结尾的网址)。

2\. 导航并进入到具体的“教学评价”操作页面。

3\. 此时网页右上角会出现一个绿色的 \*\*“🚀 自动评教”\*\* 按钮。

4\. 点击该按钮，脚本会自动开始寻找题目、勾选选项、填写评语并自动提交。

5\. 当按钮提示 \*\*“✅ 搞定”\*\* 时，即代表当前课程评教已顺利完成！



\## ⚠️ 注意事项



\* 本脚本仅供学习交流前端自动化技术使用，请合理安排评教时间，切勿恶意频繁请求服务器。

\* 使用一键评教前，建议确认评价结果是否符合你对该课程的真实意愿。

\* 如果遇到教务系统网络极度拥堵，脚本提示“超时未找到”，请刷新页面并等待加载完毕后再次点击。
